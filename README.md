# jsd2207-csmall-web-client

## Project setup项目依赖
```
npm install
```

### Compiles and hot-reloads for development运行项目
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### 引入Axios框架
````
npm i axios -s
````
#### 然后在main.js中添加配置 
```
import axios from 'axios';
Vue.prototype.axios = axios;
```
